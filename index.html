Every men's T20 international played between the ten highest-profile Full Member nations, sourced from Cricsheet's ball-by-ball archive and rebuilt into a match-and-team-level model for analysis.

The dashboard answers questions raw scorecards can't: which teams actually convert a won toss into a win, whether batting first is still the safer strategy in 2026, and how much the T20 league boom — starting with the IPL in 2008 — changed scoring rates across the international game.

Built end-to-end in Python and Power BI: parsed from raw JSON, modelled into a relational structure, and visualised across a four-page Power BI report. This page is a lightweight, browser-native version of that same dataset.
